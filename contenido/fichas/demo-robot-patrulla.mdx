Ficha temporal de demostración pensada para ocupar el escalón más alto de esta serie de pruebas.

## Objetivo

Representar una ficha más ambiciosa donde se combinan entrada, lógica y movimiento.

## Materiales

- Arduino Uno
- Chasis simple o maqueta
- Sensor
- Servo o actuador
- Cables dupont

<Activity id="act-01" title="Preparar entradas y salidas" validation="El conjunto básico responde a una condición de prueba y activa un mecanismo." />

Combina una entrada sencilla con una salida mecánica o luminosa.

<Activity id="act-02" title="Programar estados de patrulla" validation="El comportamiento pasa por varias fases identificables en orden." />

Define al menos tres estados: espera, detección y respuesta.

<Activity id="act-03" title="Demostrar el ciclo completo" validation="El alumno ejecuta el sistema completo y explica qué activa cada transición." />

Haz una demostración completa del comportamiento programado.

## Cierre

Resume qué conocimientos de fichas previas han hecho falta para completar esta prueba.
